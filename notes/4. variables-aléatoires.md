# Variables aléatoires et géométrie de L²

## Variable aléatoire

Une **variable aléatoire** à valeurs dans 𝕂 (ℝ ou ℂ) est une application mesurable X : (Ω, 𝒜, P) → (𝕂, ℬ(𝕂)), où (Ω, 𝒜, P) est un espace probabilisé. La mesurabilité garantit que les préimages de boréliens appartiennent à 𝒜 et ont donc une probabilité bien définie. Sur ℂ, une base de boréliens est donnée par les boules ouvertes : il suffit d'exiger {|X − z| < r} ∈ 𝒜 pour tout z ∈ ℂ, r > 0 — ce qui revient à demander que Re(X) et Im(X) soient simultanément mesurables. Sur ℝ, la topologie d'ordre suffit et la condition se réduit à {X ≤ t} ∈ 𝒜 pour tout t.

## L²(Ω, 𝒜, P) : espace et norme

**L² comme norme.** La norme L² d'une variable aléatoire est ‖X‖₂ = √E(|X|²). Elle appartient à la famille des normes Lᵖ : ‖X‖ₚ = E(|X|ᵖ)^(1/p). Chaque valeur de p donne une géométrie différente sur l'espace des variables aléatoires — voir *Les normes Lᵖ et pourquoi seule L² a un produit scalaire* (note à écrire).

**L² comme espace.** L²(Ω, 𝒜, P) est l'espace des variables aléatoires de carré intégrable :

L²(Ω, 𝒜, P) = { X variable aléatoire : E(|X|²) < +∞ }

(À strictement parler, les éléments de L² sont des classes d'équivalence de variables aléatoires égales presque sûrement.)

C'est un espace vectoriel sur 𝕂 (si X et Y sont dans L², alors aX + bY aussi, car |aX+bY|² ≤ 2|a|²|X|² + 2|b|²|Y|², d'où E(|aX+bY|²) ≤ 2|a|²E(|X|²) + 2|b|²E(|Y|²) < ∞). On peut montrer que cet espace est **complet** pour la norme ‖·‖₂ (théorème de Riesz-Fischer) : c'est un espace de Banach. 

**L² est le seul Lᵖ qui soit un espace de Hilbert.** Sa norme est la seule de la famille à provenir d'un produit scalaire / hermitien — ce que caractérise la règle du parallélogramme. Tous les autres Lᵖ (p ≠ 2) sont des espaces de Banach mais pas de Hilbert.

## L'espérance comme produit hermitien

Sur L²(Ω, 𝒜, P) (réel ou complexe), on définit :

⟨X, Y⟩ = E($\bar{X}\,Y$)

C'est un **produit hermitien**.

- **Sesquilinéarité** : conjugué-linéaire en X (⟨aX, Y⟩ = $\bar{a}$⟨X,Y⟩), linéaire en Y (⟨X, aY⟩ = a⟨X,Y⟩). Cela découle directement de la linéarité de l'espérance : E($\overline{aX}$·Y) = $\bar{a}$ E($\bar{X}$ Y). Sur ℝ, la conjugaison est triviale et cela se réduit à la bilinéarité.
- **Symétrie hermitienne** : ⟨X,Y⟩ = $\overline{⟨Y,X⟩}$, car E($\bar{X}$ Y) = $\overline{E(\bar{Y}X)}$. Sur ℝ, cela se réduit à la symétrie ordinaire E(XY) = E(YX).
- **Défini positif** : ⟨X,X⟩ = E(|X|²) ≥ 0, et E(|X|²) = 0 ⟹ X = 0 presque sûrement. Sur ℂ, |X|² = X·$\bar{X}$, qui est bien réel et positif ; sur ℝ, |X|² = X².

La norme associée est ‖X‖₂ = √⟨X,X⟩ = √E(|X|²). L'**orthogonalité** au sens de ce produit hermitien est ⟨X,Y⟩ = 0, c'est-à-dire E($\bar{X}$ Y) = 0.

Muni de ce produit hermitien, **L²(Ω, 𝒜, P) est un espace de Hilbert**. Tout le formalisme de la projection orthogonale — existence, unicité, caractérisation par l'orthogonalité du résidu — s'applique directement.

## Projections dans L²

L'espace L² contient des sous-espaces fermés naturels, emboîtés les uns dans les autres :

**Variables constantes** ⊂ **Variables affines en Y** ⊂ **Variables mesurables par rapport à Y**

Projeter X sur chacun de ces sous-espaces répond à la question : "quelle est la meilleure approximation de X, sachant ce qu'on connaît de Y ?"

### Projection sur les constantes : E(X)

Le sous-espace V₀ = { ω ↦ c : c ∈ 𝕂 } des variables aléatoires constantes est un sous-espace fermé de dimension 1. En identifiant une constante c à la variable constante ω ↦ c, la projection de X sur V₀ minimise ‖X − c‖₂² = E(|X − c|²). La condition d'orthogonalité du résidu exige, pour tout c₀ ∈ 𝕂 :

⟨c₀, X − c⟩ = $\bar{c}_0$·E(X − c) = 0  ⟹  **c = E(X)**

**L'espérance E(X) est la projection orthogonale de X sur l'espace des constantes.** Le résidu X − E(X) est centré et orthogonal à toute constante.

### Projection sur les variables affines en Y : la régression linéaire

Le sous-espace V₁ = { aY + b : a, b ∈ 𝕂 } est un sous-espace fermé de dimension au plus 2 (en supposant Y ∈ L² non dégénérée). La condition d'orthogonalité du résidu — que X − (a*Y + b*) soit orthogonal à tout élément de V₁, c'est-à-dire ⟨v, X − a*Y − b*⟩ = 0 pour tout v ∈ V₁ — donne le système :

- ⟨1, X − a*Y − b*⟩ = 0  ⟹  b* = E(X) − a* E(Y)
- ⟨Y, X − a*Y − b*⟩ = 0  ⟹  a* = Cov(Y, X) / Var(Y) = $\overline{\text{Cov}(X,Y)}$ / Var(Y)

où **Cov(X,Y) = E($\overline{(X-\text{E}X)}$·(Y−EY))** est la covariance complexe (hermitienne : Cov(X,Y) = $\overline{\text{Cov}(Y,X)}$) et **Var(Y) = E(|Y−EY|²)** est réelle.

Le projecteur est donc :

$$\hat{X}^{\text{lin}} = E(X) + \frac{\text{Cov}(Y,X)}{\text{Var}(Y)}\,(Y - E(Y)) = E(X) + \frac{\overline{\text{Cov}(X,Y)}}{\text{Var}(Y)}\,(Y - E(Y))$$

C'est le **meilleur prédicteur linéaire** de X par Y — obtenu comme projection orthogonale sur V₁ dans la géométrie hermitienne de L².

En notant $X_c = X-E(X)$ et $Y_c = Y-E(Y)$, on peut aussi écrire la partie centrée du projecteur sous la forme

$$\hat{X}^{\text{lin}} - E(X) = \left\langle \frac{Y_c}{\|Y_c\|_2}, X_c \right\rangle \frac{Y_c}{\|Y_c\|_2}$$

ce qui rend explicite qu'il s'agit de la projection orthogonale de $X_c$ sur la droite engendrée par $Y_c$.

 En particulier, si Cov(X,Y) = 0, alors a* = 0 et le projecteur se réduit à E(X) : l'absence de corrélation linéaire signifie exactement que connaître Y ne permet pas d'améliorer la prédiction constante.

 **Lien avec les moindres carrés sur un échantillon.** La formule a* = Cov(Y,X)/Var(Y) est l'analogue « en population » de la formule matricielle $\hat{\beta} = (\mathbf{Z}^T\mathbf{Z})^{-1}\mathbf{Z}^T\mathbf{x}$ des moindres carrés ordinaires — même principe de projection orthogonale, mais dans ℝⁿ muni du produit scalaire euclidien au lieu de L².

**Remarque de convention (statistiques/ML).** En statistique appliquée, en traitement du signal ou en machine learning, on rencontre souvent la convention hermitienne inverse, linéaire en la première variable et conjugué-linéaire en la seconde : $\langle X,Y\rangle = E(X\overline{Y})$. Avec cette convention, la même projection s'écrit naturellement avec le coefficient $\text{Cov}(X,Y)/\text{Var}(Y)$, ce qui explique la forme la plus fréquente dans ces ouvrages.

### Projection sur les fonctions de Y : E(X | Y)

Le sous-espace des variables aléatoires de L² qui dépendent seulement de Y est

$$
V_2 = \{ \text{variables de } L^2 \text{ qui dépendent seulement de } Y \}.
$$

On note E(X | Y) la variable aléatoire qui, pour chaque valeur Y = y, vaut la moyenne de X conditionnellement à Y = y. C'est une fonction de Y : elle attribue à chaque valeur possible de Y la meilleure estimation ponctuelle de X.

**Pourquoi c'est une projection.** Le résidu X − E(X | Y) a, par construction, une moyenne conditionnelle nulle dans chaque tranche Y = y. Pour tester l'orthogonalité avec un élément quelconque g(Y) ∈ V₂, on conditionne sur Y :

$$
\langle g(Y),\, X - E(X\mid Y) \rangle
= \int \overline{g(y)}\;\underbrace{E(X - E(X\mid Y) \mid Y\!=\!y)}_{=\,0}\;dP_Y(y)
= 0.
$$

Le résidu est donc orthogonal à tout V₂.

> **E(X | Y) est la projection orthogonale de X sur V₂.** C'est le meilleur prédicteur de X à partir de Y au sens de l'erreur quadratique. La régression linéaire en est le cas particulier V₁ ⊂ V₂.

En particulier, si E(X | Y) = E(X), la projection sur V₂ se réduit à la constante E(X) : aucune fonction de Y ne prédit mieux X que sa moyenne. C'est l'**indépendance en moyenne**, strictement plus faible que l'indépendance (où c'est toute la loi conditionnelle, pas seulement sa moyenne, qui ne dépend pas de Y). La hiérarchie des projections triviales est donc : Cov(X,Y) = 0 (pas de lien linéaire) ⟹ proj sur V₁ = E(X) ; E(X|Y) = E(X) (pas de lien en moyenne) ⟹ proj sur V₂ = E(X).

**Note.** La preuve complète repose sur la σ(Y)-mesurabilité (V₂ = {f(Y) : f borélienne, f(Y) ∈ L²}), la fermeture de V₂ dans L², et le théorème de projection hilbertien — voir *Espérance conditionnelle et projection : preuve complète* (note à écrire).

En particulier, si E(X | Y) = E(X), la projection sur V₂ se réduit à la constante E(X) : aucune fonction de Y ne prédit mieux X que sa moyenne. C'est l'**indépendance en moyenne**, strictement plus faible que l'indépendance.

L'**indépendance complète** $X \perp\!\!\!\perp Y$ exige davantage : non seulement X, mais **toute fonction f(X)** doit vérifier $E(f(X) | Y) = E(f(X))$. Géométriquement, cela signifie que les sous-espaces des fonctions centrées de X et de Y sont **orthogonaux** dans L² :

$$\langle f(X) - E(f(X)),\; g(Y) - E(g(Y)) \rangle = 0 \qquad \text{pour tous } f, g.$$

L'indépendance en moyenne ne teste que X contre les fonctions de Y ; l'indépendance teste toute f(X). C'est pour cela qu'une relation de type $X = Y^2 Z$ (avec Z indépendant de Y, de moyenne nulle) peut vérifier E(X|Y) = 0 = E(X) tout en rendant $X^2 = Y^4 Z^2$ entièrement prévisible à partir de Y.

La hiérarchie des projections triviales résume le tout :

- Cov(X,Y) = 0 (pas de lien linéaire) ⟹ proj sur V₁ = E(X)
- E(X|Y) = E(X) (pas de lien en moyenne) ⟹ proj sur V₂ = E(X)
- $X \perp\!\!\!\perp Y$ (indépendance) ⟹ proj de **f(X)** sur V₂ = E(f(X)) pour tout f

Chaque niveau implique le précédent, mais pas l'inverse.
