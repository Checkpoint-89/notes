# Variables aléatoires et géométrie de L²

## Variable aléatoire

Une **variable aléatoire** à valeurs dans 𝕂 (ℝ ou ℂ) est une application mesurable X : (Ω, 𝒜, P) → (𝕂, ℬ(𝕂)), où (Ω, 𝒜, P) est un espace probabilisé. La mesurabilité garantit que les préimages de boréliens appartiennent à 𝒜 et ont donc une probabilité bien définie. Sur ℂ, une base de boréliens est donnée par les boules ouvertes : il suffit d'exiger {|X − z| < r} ∈ 𝒜 pour tout z ∈ ℂ, r > 0 — ce qui revient à demander que Re(X) et Im(X) soient simultanément mesurables. Sur ℝ, la topologie d'ordre suffit et la condition se réduit à {X ≤ t} ∈ 𝒜 pour tout t.

## L²(Ω, 𝒜, P) : espace et norme

**L² comme norme.** La norme L² d'une variable aléatoire est ‖X‖₂ = √E(|X|²). Elle appartient à la famille des normes Lᵖ : ‖X‖ₚ = E(|X|ᵖ)^(1/p). Sur ℂ, |X|² = X·$\bar{X}$ = Re(X)² + Im(X)² ∈ ℝ₊ ; sur ℝ, cela se simplifie en X². Chaque valeur de p donne une géométrie différente sur l'espace des variables aléatoires — voir *Les normes Lᵖ et pourquoi seule L² a un produit scalaire*.

**L² comme espace.** L²(Ω, 𝒜, P) est l'espace des variables aléatoires de carré intégrable :

L²(Ω, 𝒜, P) = { X variable aléatoire : E(|X|²) < +∞ }

C'est un espace vectoriel sur 𝕂 (si X et Y sont dans L², alors aX + bY aussi, par |a+b|² ≤ 2|a|² + 2|b|²). Muni de la norme ‖·‖₂, il est **complet** : c'est un espace de Banach.

**L² est le seul Lᵖ qui soit un espace de Hilbert.** Sa norme est la seule de la famille à provenir d'un produit scalaire — ce que caractérise la règle du parallélogramme. Tous les autres Lᵖ (p ≠ 2) sont des espaces de Banach mais pas de Hilbert.

## L'espérance comme produit hermitien

Sur L²(Ω, 𝒜, P) (réel ou complexe), on définit :

⟨X, Y⟩ = E($\bar{X}$ ${Y}$)

C'est un **produit hermitien** — la généralisation complexe du produit scalaire. On suit la convention mathématique : conjugué-linéaire en la première variable, linéaire en la seconde (voir *Projection orthogonale et produit scalaire*).

- **Sesquilinéarité** : conjugué-linéaire en X (⟨aX, Y⟩ = $\bar{a}$⟨X,Y⟩), linéaire en Y (⟨X, aY⟩ = a⟨X,Y⟩). Cela découle directement de la linéarité de l'espérance : E($\overline{aX}$·Y) = $\bar{a}$E($\bar{X}$Y). Sur ℝ, la conjugaison est triviale et cela se réduit à la bilinéarité.
- **Symétrie hermitienne** : ⟨X,Y⟩ = $\overline{⟨Y,X⟩}$, car E($\bar{X}$Y) = $\overline{E(\bar{Y}X)}$. Sur ℝ, cela se réduit à la symétrie ordinaire E(XY) = E(YX).
- **Défini positif** : ⟨X,X⟩ = E(|X|²) ≥ 0, et E(|X|²) = 0 ⟹ X = 0 presque sûrement. Sur ℂ, |X|² = X·$\bar{X}$, qui est bien réel et positif ; sur ℝ, |X|² = X².

La norme associée est ‖X‖₂ = √⟨X,X⟩ = √E(|X|²). L'**orthogonalité** au sens de ce produit hermitien — ⟨X,Y⟩ = 0, c'est-à-dire E($\bar{X}$ Y) = 0 — est la version complexe de la **non-corrélation** : pour des variables centrées, c'est Cov(X,Y) = E($\overline{(X-\text{E}(X))}$·(Y−EY)) = 0. Sur ℝ, cela coïncide avec l'absence de corrélation linéaire.

Muni de ce produit hermitien, **L²(Ω, 𝒜, P) est un espace de Hilbert**. Tout le formalisme de la projection orthogonale — existence, unicité, caractérisation par l'orthogonalité du résidu — s'applique directement.

## Projections dans L²

L'espace L² contient des sous-espaces fermés naturels, emboîtés les uns dans les autres :

**Variables constantes** ⊂ **Variables affines en Y** ⊂ **Variables mesurables par rapport à Y**

Projeter X sur chacun de ces sous-espaces répond à la question : "quelle est la meilleure approximation de X, sachant ce qu'on connaît de Y ?"

### Projection sur les constantes : E(X)

Le sous-espace V₀ = { ω ↦ c : c ∈ 𝕂 } des variables aléatoires constantes est un sous-espace fermé de dimension 1. En identifiant une constante c à la variable constante ω ↦ c, la projection de X sur V₀ minimise ‖X − c‖₂² = E(|X − c|²). La condition d'orthogonalité du résidu exige, pour tout c₀ ∈ 𝕂 :

⟨X − c, c₀⟩ = $\bar{c_0}$·E(X − c) = 0  ⟹  **c = E(X)**

**L'espérance E(X) est la projection orthogonale de X sur l'espace des constantes.** Le résidu X − E(X) est centré et orthogonal à toute constante.

### Projection sur les variables affines en Y : la régression linéaire

Le sous-espace V₁ = { aY + b : a, b ∈ 𝕂 } est un sous-espace fermé de dimension au plus 2 (en supposant Y ∈ L² non dégénérée). La condition d'orthogonalité du résidu — que X − (a*Y + b*) soit orthogonal à tout élément de V₁ — donne le système, en développant ⟨X − a*Y − b*, ·⟩ = E($\overline{(X - a^*Y - b^*)}$ · ·) = 0 :

- ⟨X − a*Y − b*, 1⟩ = 0  ⟹  b* = E(X) − a* E(Y)
- ⟨X − a*Y − b*, Y⟩ = 0  ⟹  a* = Cov(X, Y) / Var(Y)

où **Cov(X,Y) = E($\overline{(X-\text{E}X)}$·(Y−EY))** est la covariance complexe (hermitienne : Cov(X,Y) = $\overline{\text{Cov}(Y,X)}$) et **Var(Y) = E(|Y−EY|²)** est réelle. Sur ℝ, la covariance est symétrique réelle et la formule se réduit à la régression linéaire usuelle.

Le projecteur est donc :

$$\hat{X}^{\text{lin}} = E(X) + \frac{\text{Cov}(X,Y)}{\text{Var}(Y)}\,(Y - E(Y))$$

C'est le **meilleur prédicteur linéaire** de X par Y — obtenu comme projection orthogonale sur V₁ dans la géométrie hermitienne de L².

### Projection sur les fonctions de Y : E(X | Y)

Le sous-espace des **variables mesurables par rapport à σ(Y)** est :

V₂ = { f(Y) : f borélienne, f(Y) ∈ L² }

C'est l'ensemble de toutes les fonctions de Y qui ont un moment d'ordre 2. C'est un sous-espace fermé de L² (de dimension infinie en général).

La projection orthogonale de X sur V₂ est l'unique f*(Y) ∈ V₂ telle que le résidu soit orthogonal à tout élément de V₂ :

⟨X − f*(Y), g(Y)⟩ = 0 pour tout g borélienne  ⟺  E($\overline{(X − f^*(Y))}$ · g(Y)) = 0 pour tout g borélienne

Cette condition caractérise précisément l'**espérance conditionnelle E(X | Y)** — définie comme l'unique variable σ(Y)-mesurable vérifiant E($\bar{X}$ · g(Y)) = E($\overline{\text{E}(X|Y)}$ · g(Y)) pour tout g borélien. Sur ℝ, la conjugaison disparaît et on retrouve la définition classique E((X − f*(Y)) g(Y)) = 0.

**E(X | Y) est la projection orthogonale de X sur l'espace des fonctions de Y dans L².** C'est le meilleur prédicteur de X à partir de Y — non plus contraint à être linéaire, mais autorisé à être n'importe quelle fonction mesurable. La contrainte "linéaire" devient alors un cas particulier.

## Résumé

| Sous-espace | Projecteur | Nature du résidu |
|---|---|---|
| Variables constantes | E(X) | Centré, ⊥ aux variables constantes |
| Affines en Y | E(X) + Cov(X,Y)/Var(Y) · (Y − E(Y)) | ⊥ à Y et aux constantes |
| Fonctions de Y | E(X \| Y) | ⊥ à toute f(Y) ∈ L² |

Les trois sous-espaces sont emboîtés, et la variance expliquée croît à chaque étape. La puissance prédictive augmente : les constantes ne connaissent rien de Y, la régression linéaire en exploite la corrélation, et E(X | Y) exploite toute l'information contenue dans Y.
